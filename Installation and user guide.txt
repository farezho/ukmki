1. Download dan install xampp atau laragon
Xampp: https://www.apachefriends.org/download.html
Laragon: https://laragon.org/download/index.html

2. Apabila teman2 memilih xampp, cukup klik next. Sampai muncul kotak dialog untuk memilih Apache, MySQL dll, cukup ceklis Apache dan MySQL nya saja. 
Dan apabila teman2 memilih laragon, cukup klik next dan tunggu sampai selesai

3. Letakkan folder ukmki di:
Jika menggunakan xampp >> C:/xampp/htdocs
Jika menggunakan laragon >> C:/laragon/www


[INSTALASI DATABASE]
1. Ketik localhost/phpmyadmin pada browser. Jika diminta username dan password, cukup ketik root tanpa mengisi bagian password

2. Klik menu New pada sebelah kanan, lalu ketikkan 'ukmki' (tanpa '') sebagai nama database dan klik create

3. Setelah itu klik menu import pada menu bar di bagian atas. File database berada di folder ukmki/db dengan nama file ukmki.sql
Jika sudah, klik Go pada bagian bawah tanpa mengubah settingan yang ada pada menu import tersebut


[PENGGUNAAN APLIKASI]
1. Aplikasi ini hanya bisa dijalankan di localhost (pada komputer yang terdapat aplikasi dan database ini)
2. Jalankan xampp atau laragon terlebih dahulu sebelum menggunakan aplikasi ini
3. Jika teman2 menggunakan xampp, jalankan xampp lalu klik run pada Apache dan MySQL. Tunggu sampai tombol run berubah menjadi stop
   Jika teman2 menggunakan laragon, jalankan laragon lalu klik Start All. Tunggu sampai tombol Start All berubah menjadi stop
4. Pada folder ukmki terdapat file UKMKI UNAIR, cukup klik file tersebut maka aplikasi sudah siap dijalankan
5. Masukkan username: admin dan password: secret untuk login
